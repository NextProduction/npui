Welcome to npui

Introducing npui - The component library for Next.js and React.js. Easily enhance your app development process with high-quality open components.

  • 🚀 Boost your development speed
  • 🔧 Create stunning UIs with ease
  • 🔄 Improve code quality with reusable components

Why Choose npui?

  • 🚀 High-quality components
  • 🔧 Easy to install and customize
  • 🔄 Seamless integration with Next.js and React.js
  • 🌙 Dark mode support

Getting Started

Installing npui is super easy. Just run this command:

npui add selectOption

And you're ready to go! Start enhancing your app with npui's powerful components.

Sample Usage

Usage Example 1

          
            // Code example for using the npui component
            import { Button } from 'npui';
            
            const MyComponent = () => {
              return (
                <div>
                  <Button variant="primary">Click me</Button>
                </div>
              );
            }

Usage Example 2

          
            // Another code example for using the npui component
            import { Input } from 'npui';
            
            const MyForm = () => {
              return (
                <div>
                  <Input type="text" placeholder="Enter your name"></Input>
                </div>
              );
            }
